# Vibe Coding 流程

> 这个模块用于整理如何借助 AI Agent / AI 编程工具完成一个真实项目，从想法澄清、需求文档、技术方案、任务拆解，到迭代开发、测试验收和复盘优化。

---

## 适合什么时候看

- **准备项目面试**：需要把自己的项目讲得有流程、有方法论。
- **用 AI 做项目**：不知道怎么向 AI 描述需求、拆任务、验收结果。
- **整理简历项目**：希望把“我做了什么”升级成“我如何系统化推进项目”。

---

## 核心内容

| 章节 | 你能学到什么 |
|---|---|
| Vibe Coding 标准流程 | 从想法到交付的完整 AI 辅助开发流程 |
| 需求分析 | 如何把模糊想法变成清晰 PRD |
| 技术方案 | 如何让 AI 辅助制定架构和实现路线 |
| 任务拆解 | 如何把大项目拆成可执行迭代 |
| Prompt 设计 | 如何给 AI 提供清晰上下文和约束 |
| 测试验收 | 如何判断 AI 写出来的项目是否真正完成 |
| 复盘优化 | 如何沉淀经验，形成下一轮迭代计划 |
