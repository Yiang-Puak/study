# 面试真题训练路线

> 真题训练的目标不是囤题，而是把每次被追问的空白回流到专题页。

## 一、先分清四类题

| 类型 | 训练重点 | 例子 |
| :--- | :--- | :--- |
| 八股题 | 定义、边界、原理 | RAG 完整链路是什么 |
| 工程追问 | 失败定位和取舍 | 工具超时怎么恢复 |
| 公开真题整理 | 题型覆盖和措辞变化 | 长上下文是否还需要 RAG |
| 手撕题 | 思路、复杂度和边界 | LRU、二分、滑动窗口 |

## 二、推荐训练顺序

```mermaid
graph LR
    A[专题学习] --> B[高频八股]
    B --> C[专题追问]
    C --> D[公开真题]
    D --> E[模拟口述]
    E --> A
```
