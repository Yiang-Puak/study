## 训练路线

1. 先看 [真题训练路线](真题训练路线.md)，决定从题型还是专题进入。
2. 定义和边界答不稳时刷高频题。
3. 题型和追问不熟时刷公开真题与专题题页。
4. Python 题同步补语言机制和手撕题口述。
