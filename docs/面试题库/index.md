4. 对照 [公开面试题整理](../AI%20Agent面试实践/公开面试题整理.md) 补题型。
5. 做 [综合模拟面试题库](../AI%20Agent面试实践/08_模拟面试题与答案/01_综合模拟面试题库.md)。
