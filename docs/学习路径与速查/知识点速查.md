## 工程追问速查

| 追问 | 回答抓手 |
| :--- | :--- |
| RAG 召回差怎么办？ | 拆成 query、chunk、embedding、top_k、rerank、过滤、eval 排查 |
| Agent 为什么会跑偏？ | 看目标、上下文、工具 schema、状态流转、终止条件 |
| Tool 调用失败怎么办？ | 参数校验、超时、重试、降级、人工确认、trace |
| Prompt 注入怎么防？ | 指令隔离、输入过滤、工具权限、引用边界、审计日志 |
| 怎么评估效果？ | 固定 case、自动指标、人工复核、线上 trace、失败分类 |
| 怎么降成本？ | 小模型路由、缓存、上下文裁剪、批处理、避免无效工具调用 |

## Python 工程速查

| 知识点 | 面试重点 |
| :--- | :--- |
| `asyncio` | 并发 I/O，不让检索、LLM、工具调用互相空等 |
| `pytest` | mock LLM、固定 case、回归测试 |
| logging | request_id、conversation_id、node、latency、error |
| FastAPI | Pydantic schema、异常处理、SSE、health check |
| 依赖管理 | requirements / lock / CI 一致构建 |
