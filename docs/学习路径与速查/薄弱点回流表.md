# 薄弱点回流表

> 这页是学习闭环的核心。面试题答不稳时，不要只标星；要把它回流到专题、项目和下一次口述训练。

## 使用规则

1. 每次模拟面试或刷题后，只记录最重要的 3 个薄弱点。
2. 每个薄弱点必须绑定一个专题页或项目复盘页。
3. 第二天先复述薄弱点，再继续学新内容。
4. 连续两次能 2 分钟讲清，才从“待补”移到“已掌握”。
