## 阅读约定

- 知识页先讲概念、最小示例、面试考法和练习，不把整份源码直接当正文。
- 同目录下的 `.py` 文件保留完整可运行示例，适合本地改参数、跑输出和做练习。
- 手撕题页优先看题目、思路、复杂度和追问，再打开源码页对照测试。
- Agent 岗位面试要把 Python 基础连接到工程实践：异步调用、日志、测试、依赖管理和 API 后端都要会讲。
