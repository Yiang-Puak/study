## 这个知识点解决什么问题

Python 程序要先能保存值、区分值的类型，再谈判断、循环和函数。变量负责给值起名字，数据类型决定这个值能做什么操作。

## 必会概念

| 概念 | 先记住 |
| :--- | :--- |
| 变量 | `name = value` 让名字引用一个值，重新赋值后名字可以指向别的值 |
| 类型 | 常见基础类型有 `int`、`float`、`str`、`bool`、`NoneType` |
| 命名 | 变量名可含字母、数字、下划线，不能以数字开头，建议使用蛇形命名 |
| 转换 | `int()`、`float()`、`str()`、`bool()` 用于显式转换 |

## 最小示例

```python
name = "张三"
age = 20
price = 19.9
is_ready = True
result = None

print(type(name).__name__)   # str
print(int("42") + age)       # 62
print(bool(""))              # False
```

同一个名字可以重新绑定：

```python
score = 85
score = "A"
```

这里不是“变量类型被锁死后又强行改掉”，而是名字 `score` 重新指向了另一个对象。

## 容易混淆的点

### 1. `==` 比值，`is` 比身份

```python
left = [1, 2]
right = [1, 2]

print(left == right)  # True
print(left is right)  # False
```

面试里谈对象身份时再用 `is`。判断值是否相等，优先用 `==`。

### 2. `None` 不是空字符串，也不是 `0`

`None` 表示没有结果或缺失值；空字符串表示已有一个字符串但内容为空；`0` 仍然是数字。

### 3. `bool()` 有一组常见假值

`0`、`""`、`[]`、`{}`、`None` 都会得到 `False`。业务代码里要区分“值为空”和“值不存在”时，不要只靠一个 `if value`。

## 常见考法

1. 解释 Python 变量为什么不像静态语言那样先声明类型。
2. 比较 `==` 与 `is`，并说明 `None` 判断常写成 `value is None`。
3. 判断 `bool("0")`、`bool("")`、`bool(0)` 的结果。
