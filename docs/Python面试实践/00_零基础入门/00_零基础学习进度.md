## 自检清单

每节课学完后，确保你能：

### 第1课：变量与数据类型
- [ ] 能说出变量命名的规则（字母数字下划线，不能以数字开头）
- [ ] 能区分 int、float、str、bool、None
- [ ] 能用 type() 查看变量类型
- [ ] 能做 int()/float()/str()/bool() 类型转换

### 第2课：输入输出与运算符
- [ ] 能用 print() 输出，f-string 格式化输出
- [ ] 能用 input() 获取用户输入，并转成数字
- [ ] 能说出 /、//、%、** 的区别
- [ ] 能使用 and、or、not 组合条件

### 第3课：条件判断
- [ ] 能写出 if/elif/else 结构
- [ ] 能写出嵌套 if
- [ ] 能写一行 if（三元运算符）
- [ ] 能区分 == 和 is
- [ ] 能说出哪些值会被当作 False

### 第4课：循环
- [ ] 能用 for 遍历列表、字符串、range()
- [ ] 能用 while 写条件循环
- [ ] 能正确使用 break 和 continue
- [ ] 能写嵌套循环
- [ ] 会用 enumerate() 和 zip()

### 第5课：字符串
- [ ] 能做索引和切片（包括负数索引、步长、反转）
- [ ] 会用 upper/lower/strip/replace/split/join
- [ ] 会用 f-string 格式化输出
- [ ] 知道字符串是不可变的

### 第6课：函数
- [ ] 能定义函数、调用函数
- [ ] 能写带参数和返回值的函数
- [ ] 知道默认参数的陷阱和解决方法
- [ ] 能使用 *args 和 **kwargs
- [ ] 知道局部变量和全局变量的区别
