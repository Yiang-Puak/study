# 最长不重复子串

<div class="question-sheet">
  <section>
    <h2>题目</h2>
    <p>给定字符串 <code>s</code>，返回不含重复字符的最长连续子串长度。</p>
    <p><code>"abcabcbb"</code> 的答案是 <code>3</code>，对应子串可以是 <code>"abc"</code>。</p>
  </section>
  <section>
    <h2>考察点</h2>
    <ul>
      <li>滑动窗口。</li>
      <li>哈希表记录字符最新位置。</li>
      <li>区分子串和子序列。</li>
    </ul>
  </section>
</div>

## 解题思路

窗口 `[left, right]` 始终保持“窗口内没有重复字符”：

1. 右指针逐个读字符。
2. 如果当前字符在窗口内出现过，把 `left` 跳到上次位置后一格。
3. 更新字符最新位置，再更新最长窗口长度。

关键不是重复后一点点试，而是用索引表直接跳过失效区间。

## 参考实现

```python
def length_of_longest_substring(s: str) -> int:
    last_index = {}
    left = 0
    best = 0

    for right, char in enumerate(s):
        if char in last_index and last_index[char] >= left:
            left = last_index[char] + 1

        last_index[char] = right
        best = max(best, right - left + 1)

    return best
```

## 复杂度

| 指标 | 结论 |
| :--- | :--- |
| 时间 | O(n)，每个字符被右指针处理一次 |
| 空间 | O(min(m, n))，`m` 是字符集大小 |

## 易错点

1. `last_index[char]` 可能落在旧窗口左侧，不能无条件更新 `left`。
2. 题目问的是连续子串，不是可以跳字符的子序列。
3. 边界要覆盖空字符串、全相同字符、已经全不重复三类输入。

## 面试追问

| 追问 | 回答抓手 |
| :--- | :--- |
| 为什么不是 O(n^2) | 左右指针整体只向右移动 |
| 如果要返回子串本身 | 更新最优长度时同步记录窗口起点 |
| 为什么用哈希表 | 要 O(1) 找到字符最近出现位置 |
