    СИђсђЂтЁгт╝ђжбўтъІТЋ┤уљє¶

    жбўтъІ СИ╗УдЂУђЃт»Ъ
    Context СИј Memory УЙ╣уЋї Тў»тљдУЃйтї║тѕєт«╣тЎесђЂтГўтѓетњїтЈгтЏъ
    жЋ┐С╝џУ»ЮтјІу╝Е Тў»тљдС╝џтюежбёу«ЌсђЂС┐ЮуюЪтњїТЂбтцЇС╣ІжЌ┤тЈќУѕЇ
    У«░т┐єтєЎтЁЦуГќуЋЦ Тў»тљдС╝џтї║тѕєС║Іт«ъсђЂтЂЈтЦйсђЂС║ІС╗ХтњїСИ┤ТЌХУЇЅуе┐
    У«░т┐єтЈгтЏъжћЎУ»» Тў»тљдУЃйтцёуљєтЎфтБ░сђЂУ┐ЄТюЪтњїТЮЃжЎљ
    жЋ┐С╗╗тіАТЂбтцЇ Тў»тљдуЪЦжЂЊ summaryсђЂnotesсђЂstate уџёу╗ётљѕ

    С║їсђЂуц║УїЃуюЪжбў¶

    жбў 1№╝џСИђСИфС╗БуаЂ Agent УиЉС║єтЙѕС╣Ё№╝їСИіСИІТќЄжЄїтЁеТў»тиЦтЁиу╗ЊТъю№╝їСйаТђјС╣ѕТћХтЈБ¶

    тЏъуГћжфеТъХ№╝џ

    1. тЁѕтѕєУЙетЊфС║ЏТў»тйЊтЅЇС╗ЇжюђТјеуљєуџёУДѓт»Ъ№╝їтЊфС║ЏтЈфТў»ти▓т«їТѕљТГЦжфцуџётјЪтДІУЙЊтЄ║сђѓ
    2. С┐ЮуЋЎТюђУ┐ЉтЁ│жћ«Уй«ТгАсђЂтйЊтЅЇС╗╗тіАуЏ«ТаЄсђЂу║дТЮЪсђЂтц▒У┤ЦТЌЦт┐ЌтњїТюфт«їТѕљТИЁтЇЋсђѓ
    3. т»╣ТЌЕТюЪУ┐ЄуеІтЂџ summary№╝їт»╣тЁ│жћ«у╗ЊУ«║тєЎу╗ЊТъётїќ notesсђѓ
    4. тцДтЮЌтиЦтЁиУЙЊтЄ║тГўт╝ЋућеТѕќТќЄС╗ХСйЇуй«№╝їСИЇТіітЁеТќЄТ░ИС╣ЁуЋЎтюеСИіСИІТќЄсђѓ
    5. тјІу╝Етљјућет░ЈтЏътйњТБђТЪЦуА«У«цС╗╗тіАуіХТђЂТ▓АТюЅСИбсђѓ

    жбў 2№╝џжЋ┐ТюЪУ«░т┐єтЈгтЏъС║єУ┐ЄТюЪтЂЈтЦй№╝їтЏъуГћт╝ђтДІУиЉтЂЈ№╝їТђјС╣ѕС┐«¶

    тЏъуГћжфеТъХ№╝џ

    1. у╗ЎУ«░т┐єтіаТЌХжЌ┤сђЂТЮЦТ║љсђЂуй«С┐Ат║дтњїтЈ»ТЏ┤Тќ░Тю║тѕХсђѓ
    2. тї║тѕєт╝║С║Іт«ъСИјУй»тЂЈтЦй№╝їУй»тЂЈтЦйУдЂтЁЂУ«ИУдєуЏќсђѓ
    3. тЈгтЏътЅЇтЂџућеТѕисђЂуДЪТѕитњїС╗╗тіАуЏИтЁ│У┐ЄТ╗цсђѓ
    4. тюетЏъуГћтњї Trace жЄїС┐ЮуЋЎУ«░т┐єТЮЦТ║љ№╝їСЙ┐С║јТјњТЪЦжћЎУ»»Т│етЁЦсђѓ
