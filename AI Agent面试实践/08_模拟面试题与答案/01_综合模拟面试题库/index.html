    йўҳзӣ® 1.1¶

    QпјҡAgentic Loop жҳҜд»Җд№Ҳпјҹз”»еҮәжөҒзЁӢгҖӮ

    жЁЎжӢҹзӯ”йўҳпјҲдҪ е…ҲеҸЈиҝ°пјҢеҶҚеұ•ејҖзңӢзӯ”жЎҲпјү

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡAgentic Loop жҳҜ Agent зҡ„"жҖқиҖғвҶ’иЎҢеҠЁвҶ’и§ӮеҜҹ"еҫӘзҺҜеҶізӯ–жңәеҲ¶пјҢи®© LLM иғҪеңЁд»»еҠЎжү§иЎҢиҝҮзЁӢдёӯиҝӯд»ЈжҺЁзҗҶгҖҒи°ғз”Ёе·Ҙе…·гҖҒиҺ·еҸ–еҸҚйҰҲпјҢзӣҙеҲ°д»»еҠЎе®ҢжҲҗгҖӮ 2. **дёҺзӣёдјјжҰӮеҝөеҢәеҲҶ**пјҡдёҚеҗҢдәҺеҚ•иҪ® LLM зҡ„дёҖж¬ЎжҖ§з”ҹжҲҗпјҢAgentic Loop жҳҜеӨҡиҪ®дәӨдә’пјӣдёҚеҗҢдәҺеӣәе®ҡжөҒзЁӢзҡ„и„ҡжң¬пјҢAgentic Loop дёӯжҜҸдёҖжӯҘзҡ„ Action з”ұ LLM иҮӘдё»еҶіе®ҡгҖӮ 3. **еә•еұӮеҺҹзҗҶ**пјҡжҜҸиҪ® LLM жҺҘж”¶еҪ“еүҚзҠ¶жҖҒпјҲз”ЁжҲ·иҜ·жұӮ + еҺҶеҸІеҠЁдҪң + е·Ҙе…·иҝ”еӣһпјүпјҢиҫ“еҮә ThoughtпјҲжҺЁзҗҶиҝҮзЁӢпјүе’Ң ActionпјҲе·Ҙе…·и°ғз”ЁжҲ–жңҖз»Ҳзӯ”жЎҲпјүпјҢзі»з»ҹжү§иЎҢ Action еҗҺе°Ҷ Observation иҝ”еӣһз»ҷ LLMпјҢеҫӘзҺҜзӣҙеҲ°дә§з”ҹ Final Answer жҲ–иҫҫеҲ°жңҖеӨ§иҝӯд»Јж¬Ўж•°гҖӮ 4. **еёёи§Ғеқ‘зӮ№**пјҡеҫӘзҺҜж¬Ўж•°дёҚи®ҫдёҠйҷҗдјҡеҜјиҮҙжӯ»еҫӘзҺҜе’Ң Token зҲҶзӮёпјӣLLM еҸҜиғҪдә§з”ҹж јејҸй”ҷиҜҜзҡ„ ActionпјҢйңҖиҰҒдёҘж јзҡ„и§Јжһҗе’Ңе®№й”ҷпјӣдёҖжӯҘжҺЁзҗҶй”ҷиҜҜдјҡеҜјиҮҙеҗҺз»ӯе…Ёй”ҷгҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡи®ҫзҪ® max_iterationsпјҲеҰӮ 5-10пјүпјӣе·Ҙе…·и°ғз”ЁеҠ и¶…ж—¶е’ҢйҮҚиҜ•пјӣе…ій”®иҠӮзӮ№еј•е…Ҙ Human-in-the-loopпјӣеҜ№е·Ҙе…·з»“жһңеҒҡзҪ®дҝЎеәҰиҝҮж»ӨгҖӮ **еҠ еҲҶйЎ№**пјҡиғҪз”»еҮәжөҒзЁӢеӣҫпјҲThink вҶ’ Act вҶ’ Observe еҫӘзҺҜпјҢеёҰйҖҖеҮәеҲӨж–ӯпјүгҖӮ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡAI Agent жҳҜе…·еӨҮиҮӘдё»ж„ҹзҹҘгҖҒи§„еҲ’гҖҒиЎҢеҠЁгҖҒи®°еҝҶй—ӯзҺҜиғҪеҠӣзҡ„д»»еҠЎжү§иЎҢжҷәиғҪдҪ“пјӣChatbot жҳҜеҹәдәҺ LLM зҡ„еҜ№иҜқз”ҹжҲҗзі»з»ҹгҖӮ 2. **жң¬иҙЁе·®ејӮ**пјҡChatbot зҡ„иғҪеҠӣиҫ№з•ҢжӯўдәҺ"з”ҹжҲҗж–Үжң¬"пјҢAgent зҡ„иғҪеҠӣиҫ№з•Ңжү©еұ•еҲ°"е®ҢжҲҗд»»еҠЎ"вҖ”вҖ”е®ғеҸҜд»ҘиҮӘдё»и°ғз”Ё APIгҖҒж“ҚдҪңж•°жҚ®еә“гҖҒеҶҷж–Үд»¶гҖӮ 3. **еә•еұӮж”Ҝж’‘**пјҡAgent дҫқиө– ReAct/Function Call жңәеҲ¶иҝһжҺҘеӨ–йғЁе·Ҙе…·пјҢдҫқиө–и®°еҝҶзі»з»ҹз»ҙжҢҒи·ЁдјҡиҜқзҠ¶жҖҒпјҢдҫқиө– Agentic Loop е®һзҺ°еӨҡжӯҘжҺЁзҗҶгҖӮ 4. **еёёи§Ғеқ‘зӮ№**пјҡдёҚжҳҜеҠ дәҶе·Ҙе…·зҡ„ Chatbot е°ұжҳҜ AgentпјҢе…ій”®жҳҜ"иҮӘдё»еҶізӯ–"иғҪеҠӣвҖ”вҖ”Agent иҮӘе·ұеҶіе®ҡд»Җд№Ҳж—¶еҖҷи°ғз”Ёд»Җд№Ҳе·Ҙе…·пјҢиҖҢдёҚжҳҜдәәзұ»йў„и®ҫеӣәе®ҡжөҒзЁӢгҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡAgent йңҖиҰҒе®Ңе–„зҡ„зӣ‘жҺ§пјҲи°ғз”Ёй“ҫи·ҜиҝҪиёӘпјүгҖҒе·Ҙе…·и°ғз”ЁзҪ®дҝЎеәҰйҳҲеҖјиҝҮж»ӨгҖҒејӮеёёйҷҚзә§еҲ°зәҜ LLM еҜ№иҜқгҖӮ

    йўҳзӣ® 1.5пјҲиҝӣйҳ¶пјү¶

    QпјҡMulti-Agent еҚҸдҪңжҖҺд№Ҳи®ҫи®ЎпјҹжҖҺд№ҲйҒҝе…ҚеҫӘзҺҜдҫқиө–пјҹ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡMulti-Agent жҳҜе°ҶеӨҚжқӮд»»еҠЎеҲҶи§Јз»ҷеӨҡдёӘдё“дёҡ Agent еҚҸдҪңе®ҢжҲҗпјҢжҜҸдёӘ Agent жңүзӢ¬з«Ӣзҡ„и§’иүІгҖҒе·Ҙе…·йӣҶе’Ңи®°еҝҶгҖӮ 2. **и®ҫи®ЎиҰҒзҙ **пјҡи§’иүІеҲҶе·ҘпјҲз ”з©¶е‘ҳ/еҶҷжүӢ/е®Ўж ёе‘ҳпјүгҖҒйҖҡдҝЎжңәеҲ¶пјҲж¶ҲжҒҜдј йҖ’/е…ұдә«зҠ¶жҖҒжұ /дё»д»ҺеҚҸи°ғпјүгҖҒд»»еҠЎзј–жҺ’пјҲPipeline/еӣҫз»“жһ„пјүгҖӮ 3. **йҒҝе…ҚеҫӘзҺҜдҫқиө–**пјҡйҖҡиҝҮдёӯеӨ®еҚҸи°ғеҷЁпјҲOrchestratorпјүжЁЎејҸпјҢAgent д№Ӣй—ҙдёҚзӣҙжҺҘйҖҡдҝЎпјҢжүҖжңүж¶ҲжҒҜйғҪз»ҸиҝҮеҚҸи°ғеҷЁиҪ¬еҸ‘пјҢз”ұеҚҸи°ғеҷЁжҺ§еҲ¶жөҒзЁӢе’ҢзҠ¶жҖҒеҗҢжӯҘгҖӮ 4. **еёёи§Ғеқ‘зӮ№**пјҡзҠ¶жҖҒдёҖиҮҙжҖ§й—®йўҳпјҲеӨҡдёӘ Agent еҗҢж—¶дҝ®ж”№е…ұдә«зҠ¶жҖҒпјүпјӣеҫӘзҺҜи°ғз”ЁжҺ§еҲ¶пјҲA и°ғ BпјҢB и°ғ Aпјүпјӣй”ҷиҜҜдј ж’ӯпјҲдёҖдёӘ Agent еӨұиҙҘеҜјиҮҙе…Ёй“ҫи·ҜеӨұиҙҘпјүгҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡLangGraph зҡ„ StateGraph жҳҫејҸз®ЎзҗҶзҠ¶жҖҒе’ҢжөҒиҪ¬пјӣжҜҸдёӘ Agent зӢ¬з«Ӣзҡ„й”ҷиҜҜеӨ„зҗҶе’ҢйҷҚзә§зӯ–з•Ҙпјӣеј•е…Ҙи¶…ж—¶е’ҢзҶ”ж–ӯжңәеҲ¶гҖӮ

    дәҢгҖҒи®°еҝҶзі»з»ҹпјҲвӯҗвӯҗвӯҗвӯҗпјү¶

    йўҳзӣ® 2.1¶

    QпјҡAgent зҡ„зҹӯжңҹи®°еҝҶе’Ңй•ҝжңҹи®°еҝҶжңүд»Җд№ҲеҢәеҲ«пјҹ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡзҹӯжңҹи®°еҝҶжҳҜдјҡиҜқзә§еҲ«зҡ„еҜ№иҜқеҺҶеҸІпјҢй•ҝжңҹи®°еҝҶжҳҜи·ЁдјҡиҜқжҢҒд№…еҢ–зҡ„зҹҘиҜҶ/еҒҸеҘҪеӯҳеӮЁгҖӮ 2. **жң¬иҙЁеҢәеҲ«**пјҡзҹӯжңҹи®°еҝҶе…іжіЁ"иҝҷж¬ЎеҜ№иҜқиҜҙдәҶд»Җд№Ҳ"пјҢй•ҝжңҹи®°еҝҶе…іжіЁ"з”ЁжҲ·й•ҝжңҹеҒҸеҘҪ/еҺҶеҸІдәӢд»¶жҳҜд»Җд№Ҳ"гҖӮ 3. **е®һзҺ°жңәеҲ¶**пјҡзҹӯжңҹи®°еҝҶз”Ё Python list з»ҙжҠӨ message еҲ—иЎЁпјҢдј е…Ҙ LLM ж—¶еҸ— max_tokens йҷҗеҲ¶пјҢи¶…йҷҗж—¶з”Ёж»‘еҠЁзӘ—еҸЈжҲ–ж‘ҳиҰҒеҺӢзј©пјӣй•ҝжңҹи®°еҝҶе…ій”®дҝЎжҒҜжҸҗеҸ–еҗҺ Embedding еҗ‘йҮҸеҢ–пјҢеӯҳе…Ҙ Chroma/Milvus зӯүеҗ‘йҮҸеә“гҖӮ 4. **еёёи§Ғеқ‘зӮ№**пјҡзҹӯжңҹи®°еҝҶдёҚеҒҡеҺӢзј©дјҡеҜјиҮҙ Token жҡҙж¶ЁгҖҒAPI иҙ№з”ЁжҝҖеўһпјӣй•ҝжңҹи®°еҝҶжЈҖзҙўеҸ¬еӣһеҸҜиғҪжјҸжҺүе…ій”®дҝЎжҒҜгҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡзҹӯжңҹи®°еҝҶйҮҮз”Ё"Buffer + Summary"ж··еҗҲзӯ–з•Ҙпјӣй•ҝжңҹи®°еҝҶеҒҡе®ҡжңҹжё…зҗҶ/жӣҙж–°пјҲTTLпјүпјӣеӨҡз”ЁжҲ·еңәжҷҜдёӢи®°еҝҶеҝ…йЎ»йҡ”зҰ»пјҲuser_id еҒҡеҲҶеҢәпјүгҖӮ

    йўҳзӣ® 2.2¶

    QпјҡToken йҷҗеҲ¶дёӢжҖҺд№ҲеҺӢзј©и®°еҝҶпјҹ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡToken еҺӢзј©жҳҜжҢҮеңЁ LLM дёҠдёӢж–Үй•ҝеәҰеҸ—йҷҗзҡ„жғ…еҶөдёӢпјҢз”Ёжӣҙе°‘ Token дҝқз•ҷжӣҙеӨҡжңүж•ҲдҝЎжҒҜгҖӮ 2. **дёәд»Җд№Ҳеҝ…йЎ»еҒҡ**пјҡAgent еӨҡиҪ®еҫӘзҺҜеҗҺеҺҶеҸІж¶ҲжҒҜзәҝжҖ§еўһй•ҝпјҢеҫҲеҝ«е°ұдјҡи§ҰеҸҠжЁЎеһӢдёҠдёӢж–ҮдёҠйҷҗгҖӮ 3. **еҺӢзј©ж–№жЎҲ**пјҡж»‘еҠЁзӘ—еҸЈпјҲдҝқз•ҷжңҖиҝ‘ N иҪ®пјүгҖҒж‘ҳиҰҒжӣҝжҚўпјҲж—©жңҹеҜ№иҜқз”ҹжҲҗж‘ҳиҰҒпјүгҖҒе®һдҪ“жҸҗеҸ–пјҲз»“жһ„еҢ–еӯҳеӮЁе…ій”®дҝЎжҒҜпјүгҖҒеҗ‘йҮҸеҢ–жЈҖзҙўпјҲе…ЁйғЁеӯҳе…Ҙ VectorDBпјҢжҢүйңҖеҸ¬еӣһпјүгҖӮ 4. **еёёи§Ғеқ‘зӮ№**пјҡж‘ҳиҰҒз”ҹжҲҗжң¬иә«ж¶ҲиҖ— Token е’Ңж—¶й—ҙпјӣж»‘еҠЁзӘ—еҸЈдјҡдёўеӨұж—©жңҹе…ій”®дҝЎжҒҜпјӣеҗ‘йҮҸжЈҖзҙўеҸҜиғҪеҸ¬еӣһдёҚзӣёе…іеҶ…е®№гҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡж··еҗҲзӯ–з•ҘвҖ”вҖ”иҝ‘жңҹ 5 иҪ®дҝқз•ҷеҺҹе§Ӣж¶ҲжҒҜпјҢжӣҙж—©зҡ„з”ҹжҲҗж‘ҳиҰҒпјӣж‘ҳиҰҒз”ҹжҲҗз”Ёдҫҝе®ңзҡ„е°ҸжЁЎеһӢпјӣзӣ‘жҺ§дёҠдёӢж–Ү Token дҪҝз”ЁйҮҸгҖӮ

    йўҳзӣ® 2.3пјҲиҝӣйҳ¶пјү¶

    QпјҡеӨ§йҮҸи®°еҝҶжҖҺд№ҲеӨ„зҗҶпјҹеҰӮдҪ•йҒҝе…ҚдҝЎжҒҜиҝҮиҪҪпјҹ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡеӨ§йҮҸи®°еҝҶеӨ„зҗҶжҳҜжҢҮеңЁ Agent иҝҗиЎҢиҝҮзЁӢдёӯпјҢеҺҶеҸІдҝЎжҒҜи§„жЁЎиҶЁиғҖж—¶зҡ„еӯҳеӮЁгҖҒжЈҖзҙўе’ҢзӯӣйҖүзӯ–з•ҘгҖӮ 2. **й—®йўҳжң¬иҙЁ**пјҡдҝЎжҒҜиҝҮиҪҪеҜјиҮҙ LLM жіЁж„ҸеҠӣеҲҶж•ЈпјҲеӨ§жө·жҚһй’Ҳй—®йўҳпјүпјҢеҗҢж—¶ Token жҲҗжң¬е’Ңе»¶иҝҹжҡҙж¶ЁгҖӮ 3. **еӨ„зҗҶж–№жЎҲ**пјҡеҲҶеұӮеӯҳеӮЁпјҲзғӯж•°жҚ®еҶ…еӯҳвҶ’жё©ж•°жҚ® RedisвҶ’еҶ·ж•°жҚ®еҗ‘йҮҸеә“пјүгҖҒжЈҖзҙўејҸеҸ¬еӣһпјҲж №жҚ®еҪ“еүҚ query еҒҡиҜӯд№үжЈҖзҙўпјҢеҸӘеҸ¬еӣһ top_kпјүгҖҒйҮҚиҰҒжҖ§иҜ„еҲҶгҖҒж—¶й—ҙиЎ°еҮҸгҖӮ 4. **еёёи§Ғеқ‘зӮ№**пјҡжЈҖзҙўеҸ¬еӣһеҸҜиғҪйҒ—жјҸи·Ёж—¶й—ҙе…іиҒ”зҡ„дҝЎжҒҜпјӣйҮҚиҰҒжҖ§иҜ„еҲҶи§„еҲҷйҡҫи®ҫи®Ўпјӣж—¶й—ҙиЎ°еҮҸеҸҜиғҪиҜҜеҲ йҮҚиҰҒдҝЎжҒҜгҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡеј•е…Ҙе…іиҒ”еӣҫи®©зӣёе…іи®°еҝҶиҮӘеҠЁе…іиҒ”пјӣе®ҡжңҹеҒҡи®°еҝҶж•ҙзҗҶпјӣз”ЁжҲ·дё»еҠЁж Үи®°"йҮҚиҰҒ"зҡ„и®°еҝҶж°ёдёҚиҝҮжңҹгҖӮ

    дёүгҖҒRAG жЈҖзҙўеўһејәпјҲвӯҗвӯҗвӯҗвӯҗвӯҗ еҝ…иҖғпјү¶

    йўҳзӣ® 3.1¶

    QпјҡRAG зҡ„е®Ңж•ҙжөҒзЁӢжҳҜд»Җд№Ҳпјҹз”»еҮәй“ҫи·ҜеӣҫгҖӮ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡRAGпјҲRetrieval-Augmented Generationпјү= жЈҖзҙў + еўһејә + з”ҹжҲҗпјҢи®© LLM еңЁеӣһзӯ”еүҚе…ҲжҹҘиө„ж–ҷпјҢеҮҸе°‘е№»и§үгҖӮ 2. **дёҺзәҜ LLM зҡ„еҢәеҲ«**пјҡзәҜ LLM дҫқиө–и®ӯз»ғеҸӮж•°дёӯзҡ„зҹҘиҜҶпјҲжңүжҲӘжӯўж—ҘжңҹгҖҒдјҡе№»и§үпјүпјӣRAG и®© LLM еҠЁжҖҒи®ҝй—®еӨ–йғЁзҹҘиҜҶеә“пјҢзӯ”жЎҲжӣҙе®һж—¶гҖҒеҸҜжәҜжәҗгҖӮ 3. **еӣӣйҳ¶ж®өжөҒзЁӢ**пјҡзҙўеј•пјҲж–ҮжЎЈвҶ’ChunkвҶ’EmbeddingвҶ’еҗ‘йҮҸеә“пјүгҖҒжЈҖзҙўпјҲQuery EmbeddingвҶ’зӣёдјјеәҰжҗңзҙўвҶ’еҸ¬еӣһ top_kпјүгҖҒеўһејәпјҲе°Ҷ chunk жӢје…Ҙ PromptпјүгҖҒз”ҹжҲҗпјҲLLM з»“еҗҲдёҠдёӢж–Үз”ҹжҲҗзӯ”жЎҲпјүгҖӮ 4. **еёёи§Ғеқ‘зӮ№**пјҡChunk еҲҮеҲҶдёҚеҪ“еҜјиҮҙиҜӯд№үеүІиЈӮпјӣжЈҖзҙўеҸ¬еӣһдёҚзӣёе…іж–ҮжЎЈдјҡжұЎжҹ“з”ҹжҲҗз»“жһңпјӣдёҠдёӢж–ҮиҝҮй•ҝи¶…еҮәжЁЎеһӢйҷҗеҲ¶гҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡж··еҗҲжЈҖзҙўпјҲиҜӯд№ү+е…ій”®иҜҚпјү+ Reranker зІҫжҺ’ + еӨҡзә§зј“еӯҳ + жЈҖзҙўз»“жһңжәҜжәҗгҖӮ

    йўҳзӣ® 3.2¶

    QпјҡEmbedding жҖҺд№ҲйҖүпјҹChunk жҖҺд№ҲеҲҮпјҹ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡEmbedding жҳҜе°Ҷж–Үжң¬жҳ е°„еҲ°иҜӯд№үеҗ‘йҮҸзҡ„иҝҮзЁӢпјҢChunk жҳҜе°Ҷй•ҝж–ҮжЎЈеҲҮеҲҶдёәйҖӮеҗҲжЈҖзҙўе’Ңз”ҹжҲҗзҡ„зүҮж®өгҖӮ 2. **йҖүеһӢеҺҹеҲҷ**пјҡдёӯж–ҮйҖҡз”Ё BGE-M3/GTE-largeпјӣиӢұж–Ү E5/OpenAIпјӣдё“дёҡйўҶеҹҹеҝ…йЎ»еңЁйўҶеҹҹиҜӯж–ҷдёҠеҫ®и°ғгҖӮ 3. **Chunk зӯ–з•Ҙ**пјҡеӣәе®ҡй•ҝеәҰпјҲз®ҖеҚ•дҪҶжңүиҜӯд№үеүІиЈӮпјүгҖҒжҢүж®өиҗҪ/еҸҘеӯҗпјҲдҝқз•ҷиҮӘз„¶иҫ№з•ҢпјүгҖҒйҮҚеҸ зӘ—еҸЈпјҲеҮҸе°‘иҫ№з•ҢдҝЎжҒҜдёўеӨұпјүгҖҒиҜӯд№үеҲҮеҲҶпјҲж•ҲжһңеҘҪдҪҶжҲҗжң¬й«ҳпјүгҖӮ 4. **еёёи§Ғеқ‘зӮ№**пјҡChunk еӨӘеӨ§вҶ’жЈҖзҙўзІ’еәҰзІ—пјӣChunk еӨӘе°ҸвҶ’зјәе°‘дёҠдёӢж–ҮпјҢLLM з”ҹжҲҗиҙЁйҮҸе·®гҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡеӨҡзІ’еәҰзҙўеј•пјҲеҸҘеӯҗзә§+ж®өиҗҪзә§пјүпјӣChunk еҶ…еҠ е…Ҙж Үйўҳ/з« иҠӮдҝЎжҒҜпјӣеҜ№й«ҳйў‘жҹҘиҜўеҒҡ Embedding з»“жһңзј“еӯҳгҖӮ

    йўҳзӣ® 3.3¶

    Qпјҡд»Җд№ҲжҳҜж··еҗҲжЈҖзҙўпјҹОұ жҖҺд№Ҳи°ғпјҹ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡж··еҗҲжЈҖзҙў = иҜӯд№үжЈҖзҙўпјҲеҗ‘йҮҸзӣёдјјеәҰпјү+ е…ій”®иҜҚжЈҖзҙўпјҲBM25пјүпјҢдёӨиҖ…з»“жһңиһҚеҗҲеҗҺиҝ”еӣһгҖӮ 2. **дёәд»Җд№ҲйңҖиҰҒ**пјҡзәҜиҜӯд№үжЈҖзҙўеҜ№дё“дёҡжңҜиҜӯеҢ№й…ҚдёҚеҮҶпјӣзәҜе…ій”®иҜҚжЈҖзҙўж— жі•зҗҶи§ЈеҗҢд№үиҜҚгҖӮ 3. **иһҚеҗҲе…¬ејҸ**пјҡ`score = Оұ * vector_score + (1-Оұ) * bm25_score` 4. **Оұ еҠЁжҖҒи°ғж•ҙ**пјҡе®һдҪ“жҹҘиҜўпјҲжҹҘдәәеҗҚ/жі•жқЎпјүОұ еҒҸе°ҸпјҲ0.2-0.4пјүпјӣеҲҶжһҗжҹҘиҜўпјҲжҖ»з»“/и§ЈйҮҠпјүОұ еҒҸеӨ§пјҲ0.6-0.8пјүпјӣеҸҜз”Ёе°ҸеҲҶзұ»еҷЁеҲӨж–ӯ query зұ»еһӢиҮӘеҠЁи·Ҝз”ұгҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡи·Ҝз”ұеҲҶзұ»еҗҺеҶіе®ҡ ОұпјӣиһҚеҗҲеҗҺ Reranker зІҫжҺ’пјӣи®°еҪ•дёҚеҗҢ Оұ дёӢз”ЁжҲ·ж»Ўж„ҸеәҰпјҢA/B жөӢиҜ•и°ғдјҳгҖӮ

    йўҳзӣ® 3.4пјҲиҝӣйҳ¶пјү¶

    QпјҡReranker жҳҜд»Җд№Ҳпјҹдёәд»Җд№ҲйңҖиҰҒдёӨйҳ¶ж®өжЈҖзҙўпјҹ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡReranker жҳҜжЈҖзҙўеҗҺзҡ„зІҫжҺ’жЁЎеһӢпјҢеҜ№еҸ¬еӣһзҡ„еӨ§йҮҸеҖҷйҖүз»“жһңйҮҚж–°жү“еҲҶжҺ’еәҸгҖӮ 2. **дёәд»Җд№ҲйңҖиҰҒдёӨйҳ¶ж®ө**пјҡз¬¬дёҖйҳ¶ж®өпјҲеҸ¬еӣһпјүиҰҒжұӮйҖҹеәҰеҝ«пјҢз”ЁиҪ»йҮҸж–№жі•еҸ¬еӣһ top_100пјӣз¬¬дәҢйҳ¶ж®өпјҲзІҫжҺ’пјүиҰҒжұӮзІҫеҮҶеәҰй«ҳпјҢз”Ёжӣҙејәзҡ„дәӨеҸүзј–з ҒеҷЁзІҫз»Ҷжү“еҲҶпјҢеҸ– top_5/10гҖӮеҰӮжһңзӣҙжҺҘз”ЁзІҫжҺ’еҒҡеҸ¬еӣһпјҢи®Ўз®—йҮҸеӨӘеӨ§гҖӮ 3. **е…ёеһӢжЁЎеһӢ**пјҡbge-rerankerгҖҒCohere RerankгҖҒJina Reranker 4. **еёёи§Ғеқ‘зӮ№**пјҡReranker д№ҹжңүе№»и§үйЈҺйҷ©вҖ”вҖ”з»ҷдёҚзӣёе…іж–ҮжЎЈжү“й«ҳеҲҶпјӣзІҫжҺ’жЁЎеһӢиҫ“е…Ҙй•ҝеәҰжңүйҷҗпјҢй•ҝж–ҮжЎЈйңҖиҰҒжҲӘж–ӯгҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡReranker з»“жһңеҠ е…ҘзҪ®дҝЎеәҰйҳҲеҖјпјӣReranker з»“жһңд№ҹзј“еӯҳпјӣеҜ№й«ҳйў‘ query зҡ„зІҫжҺ’з»“жһңеҒҡзҰ»зәҝйў„и®Ўз®—гҖӮ

    еӣӣгҖҒLangChain / LangGraphпјҲвӯҗвӯҗвӯҗвӯҗпјү¶

    йўҳзӣ® 4.1¶

    QпјҡLangChain е’Ң LangGraph жңүд»Җд№ҲеҢәеҲ«пјҹд»Җд№Ҳж—¶еҖҷз”Ёе“ӘдёӘпјҹ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡLangChain жҳҜеӨ§жЁЎеһӢеә”з”ЁејҖеҸ‘жЎҶжһ¶пјӣLangGraph жҳҜ LangChain з”ҹжҖҒзҡ„жү©еұ•пјҢдё“й—Ёи§ЈеҶіеӨҚжқӮе·ҘдҪңжөҒе’ҢеӨҡ Agent еҚҸдҪңзҡ„еӣҫзј–жҺ’й—®йўҳгҖӮ 2. **жң¬иҙЁеҢәеҲ«**пјҡLangChain йҖӮеҗҲ"зәҝжҖ§з®ЎйҒ“"пјҲAвҶ’BвҶ’CпјүпјҢLangGraph йҖӮеҗҲ"еӨҚжқӮжөҒзЁӢеӣҫ"пјҲеҲҶж”ҜгҖҒеҫӘзҺҜгҖҒеӨҡиҠӮзӮ№еҚҸдҪңпјүгҖӮ 3. **еә•еұӮжңәеҲ¶**пјҡLangChain з”Ё LCEL зҡ„ `|` дёІиҒ”з»„д»¶пјҢж•°жҚ®йҡҗејҸжөҒиҪ¬пјӣLangGraph жҳҫејҸе®ҡд№ү StateгҖҒNodesгҖҒEdgesпјҢж”ҜжҢҒжқЎд»¶еҲҶж”Ҝе’ҢеҫӘзҺҜгҖӮ 4. **йҖүжӢ©еҺҹеҲҷ**пјҡз®ҖеҚ• RAG/еҚ•иҪ®е·Ҙе…·и°ғз”Ёз”Ё LangChainпјӣеӨҡ Agent еҚҸдҪң/еҫӘзҺҜиҝӯд»Ј/е®Ўжү№жөҒз”Ё LangGraphгҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡLangChain з”Ё `astream_events` еҒҡе®һж—¶зӣ‘жҺ§пјӣLangGraph з”Ё `checkpoint` еҒҡж–ӯзӮ№з»ӯдј е’ҢзҠ¶жҖҒеӣһж»ҡгҖӮ

    йўҳзӣ® 4.2пјҲиҝӣйҳ¶пјү¶

    QпјҡLangGraph зҡ„зҠ¶жҖҒз®ЎзҗҶжҳҜжҖҺд№Ҳе®һзҺ°зҡ„пјҹ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ 1. **ж ёеҝғе®ҡд№ү**пјҡLangGraph з”ЁжҳҫејҸзҡ„ `State` еҜ№иұЎз®ЎзҗҶе·ҘдҪңжөҒдёӯзҡ„е…ұдә«ж•°жҚ®пјҢState жҳҜдёҖдёӘ TypedDict жҲ– Pydantic BaseModelгҖӮ 2. **дёәд»Җд№ҲйңҖиҰҒ**пјҡеӨҡ Agent еҚҸдҪңж—¶пјҢжҜҸдёӘ Agent йңҖиҰҒиҜ»еҸ–е…¶д»– Agent зҡ„иҫ“еҮәпјҢйҡҗејҸдј йҖ’е®№жҳ“еҮәй”ҷпјҢжҳҫејҸ State и®©ж•°жҚ®жөҒжё…жҷ°еҸҜи§ҒгҖӮ 3. **е®һзҺ°жңәеҲ¶**пјҡе®ҡд№ү State з»“жһ„пјҲеҢ…еҗ« messages/documents/next_step зӯүеӯ—ж®өпјүпјӣиҠӮзӮ№еҮҪж•°зӯҫеҗҚ `def agent_node(state: State) -> State:`пјӣжҷ®йҖҡиҫ№еӣәе®ҡжөҒиҪ¬пјҢжқЎд»¶иҫ№йҖҡиҝҮ router еҮҪж•°ж №жҚ® State еҶіе®ҡдёӢдёҖжӯҘгҖӮ 4. **еёёи§Ғеқ‘зӮ№**пјҡState жҳҜеӯ—е…ёеј•з”ЁпјҢеҺҹең°дҝ®ж”№еҸҜиғҪеҜјиҮҙеүҜдҪңз”ЁпјӣиҠӮзӮ№й—ҙдј йҖ’еӨ§еҜ№иұЎеҜјиҮҙеҶ…еӯҳејҖй”ҖпјӣеҫӘзҺҜиҫ№йңҖиҰҒи®ҫзҪ®йҖҖеҮәжқЎд»¶еҗҰеҲҷжӯ»еҫӘзҺҜгҖӮ 5. **з”ҹдә§дјҳеҢ–**пјҡState з”Ё Pydantic еҒҡж ЎйӘҢпјӣеӨ§еҜ№иұЎеӯҳеј•з”ЁиҖҢйқһеҶ…е®№пјӣжҺҘе…Ҙ checkpointer е®һзҺ°ж–ӯзӮ№з»ӯдј пјӣз”Ё interrupt е®һзҺ° Human-in-the-loopгҖӮ

    дә”гҖҒз»јеҗҲеңәжҷҜйўҳпјҲвӯҗвӯҗвӯҗвӯҗвӯҗ еӨ§еҺӮзҲұиҖғпјү¶

    йўҳзӣ® 5.1¶

    Qпјҡи®ҫи®ЎдёҖдёӘз”өе•Ҷе®ўжңҚ AgentпјҢйңҖиҰҒе“ӘдәӣжЁЎеқ—пјҹжҖҺд№Ҳе®һзҺ°пјҹ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ **жһ¶жһ„и®ҫи®Ў**пјҡ 1. **ж„ҸеӣҫиҜҶеҲ«жЁЎеқ—**пјҡз”Ё LLM жҲ–еҲҶзұ»еҷЁеҲӨж–ӯз”ЁжҲ·ж„ҸеӣҫпјҲжҹҘзү©жөҒ/йҖҖжҚўиҙ§/е•Ҷе“Ғе’ЁиҜў/жҠ•иҜүпјү 2. **жЈҖзҙўжЁЎеқ—**пјҡRAG жЈҖзҙўе•Ҷе“Ғ FAQгҖҒи®ўеҚ•дҝЎжҒҜгҖҒзү©жөҒзҠ¶жҖҒ 3. **е·Ҙе…·жЁЎеқ—**пјҡжҹҘи®ўеҚ• APIгҖҒз”іиҜ·йҖҖж¬ҫ APIгҖҒиҪ¬дәәе·Ҙ API 4. **и®°еҝҶжЁЎеқ—**пјҡи®°дҪҸз”ЁжҲ·еҺҶеҸІи®ўеҚ•гҖҒеҒҸеҘҪгҖҒжҠ•иҜүи®°еҪ• 5. **з”ҹжҲҗжЁЎеқ—**пјҡз»“еҗҲжЈҖзҙўз»“жһңе’Ңи®°еҝҶз”ҹжҲҗеӣһеӨҚ **е®һзҺ°ж–№жЎҲ**пјҡ - LangChain еҒҡ RAG е’Ңе·Ҙе…·и°ғз”Ё - LangGraph зј–жҺ’еӨҚжқӮжөҒзЁӢпјҲеҰӮ"жҠ•иҜүвҶ’е®Ўж ёвҶ’йҖҖж¬ҫ"йңҖиҰҒеӨҡиҠӮзӮ№е®Ўжү№пјү - еҗ‘йҮҸеә“еӯҳе•Ҷе“ҒзҹҘиҜҶе’ҢеҺҶеҸІеҜ№иҜқ - Redis зј“еӯҳзғӯзӮ№й—®йўҳзӯ”жЎҲ **з”ҹдә§иҰҒзӮ№**пјҡ - ж•Ҹж„ҹж“ҚдҪңпјҲйҖҖж¬ҫ/ж”№ең°еқҖпјүйңҖиҰҒ Human-in-the-loop зЎ®и®Ө - жҺҘе…Ҙзӣ‘жҺ§з»ҹи®Ўи§ЈеҶізҺҮгҖҒе№іеқҮе“Қеә”ж—¶й—ҙгҖҒз”ЁжҲ·ж»Ўж„ҸеәҰ - ж— жі•и§ЈеҶізҡ„й—®йўҳиҮӘеҠЁиҪ¬дәәе·ҘпјҢе№¶жҗәеёҰе®Ңж•ҙдёҠдёӢж–Ү

    йўҳзӣ® 5.2пјҲиҝӣйҳ¶пјү¶

    QпјҡдҪ зҡ„ RAG йЎ№зӣ®жЈҖзҙўж•ҲжһңдёҚзҗҶжғіпјҢжҖҺд№ҲжҺ’жҹҘе’ҢдјҳеҢ–пјҹ

    зӮ№еҮ»жҹҘзңӢеҸӮиҖғзӯ”жЎҲ **жҺ’жҹҘжҖқи·ҜпјҲд»Һиҝ‘еҲ°иҝңпјү**пјҡ 1. **Prompt еұӮ**пјҡжЈҖжҹҘдёҠдёӢж–ҮжҳҜеҗҰжӢје…ҘжӯЈзЎ®пјҹжҳҜеҗҰжңү"еҝҪз•ҘеҸӮиҖғиө„ж–ҷ"зҡ„жҸҗзӨәжі„жјҸпјҹ 2. **з”ҹжҲҗеұӮ**пјҡLLM жҳҜеҗҰзҗҶи§ЈдәҶдёҠдёӢж–ҮпјҹжҚўжӣҙејәзҡ„жЁЎеһӢжөӢиҜ• 3. **жЈҖзҙўеұӮ**пјҡеҸ¬еӣһзҡ„ж–ҮжЎЈжҳҜеҗҰдёҺ query зӣёе…іпјҹHit@K еӨҡе°‘пјҹ 4. **зҙўеј•еұӮ**пјҡChunk еҲҮеҲҶжҳҜеҗҰеҗҲзҗҶпјҹEmbedding иҙЁйҮҸеҰӮдҪ•пјҹ 5. **ж•°жҚ®еұӮ**пјҡзҹҘиҜҶеә“жҳҜеҗҰеҢ…еҗ«зӯ”жЎҲпјҹж•°жҚ®иҙЁйҮҸеҰӮдҪ•пјҹ **дјҳеҢ–ж–№жЎҲ**пјҡ - жЈҖзҙўеұӮпјҡеҠ  BM25 ж··еҗҲжЈҖзҙў + Reranker зІҫжҺ’ + жҹҘиҜўж”№еҶҷ - зҙўеј•еұӮпјҡдјҳеҢ– Chunk зӯ–з•ҘгҖҒеҫ®и°ғ EmbeddingгҖҒеҠ ж–ҮжЎЈж Үйўҳ/ж Үзӯҫ - ж•°жҚ®еұӮпјҡиЎҘе……зјәеӨұж–ҮжЎЈгҖҒжё…жҙ—дҪҺиҙЁйҮҸж•°жҚ® - зі»з»ҹеұӮпјҡеҠ еӨҡзә§зј“еӯҳгҖҒејӮжӯҘзҙўеј•жӣҙж–°гҖҒжЈҖзҙўз»“жһңжәҜжәҗ
