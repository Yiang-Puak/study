    AI Agent 工程学习手册¶

    面向两件事：知识点学习 和 面试题训练。
    先从专题入口建立概念，再用高频题、公开真题和工程追问检查自己是否真的会答。

    主入口¶

    入口 适合什么时候看 进入
    Agent 知识点 系统学习 LLM、Context、RAG、Tool、Workflow、Eval、Safety 进入 Agent 知识点
    面试题库 直接刷高频题、公开真题、工程追问和模拟面试 打开面试题库
    Python 知识点 补 Python 语言基础、数据结构、语言机制和手撕题 进入 Python 知识点
    项目与代码实践 用代码实践、项目复盘和工程案例支撑面试表达 查看实践入口
    路线与速查 用雷达、自测、GitHub 项目和薄弱点回流安排复习 打开路线与速查

    知识点入口¶

    方向 核心入口 面试重点
    LLM 与 Token AI 底层框架全景图 输入输出、Token、上下文边界
    Context 与 Memory Context 工程专题 窗口预算、压缩、记忆与检索
    RAG RAG 检索工程专题 Chunk、召回、Rerank、引用、评测
    Tool 与 MCP Tool Calling 与 MCP 专题 Schema、运行时边界、协议、安全
    Workflow 与 LangGraph 编排专题 State、分支、循环、Multi-Agent 取舍
    Eval 与 Safety 治理专题 Eval、Trace、护栏、HITL
    Prompt 工程 Prompt 专题 结构化输出、parser、prompt eval
    前沿趋势 前沿与趋势 Agentic RAG、MCP、多 Agent、成本与安全
    Python Python 知识点总览 语言机制、容器、异常、手撕题

    面试题入口¶

    题型 入口 用途
    高频题 Agent 高频题总览 快速压缩定义、边界、取舍和工程回答
    公开真题 公开面试题整理 熟悉公开题型和追问措辞
    工程追问 工程追问题库 训练方案设计、故障排查和取舍表达
    Python 真题 Python 面试真题整理 复习语言机制、数据结构和手撕题口述
    模拟面试 综合模拟面试题库 把知识点串成完整回答

    实践补充¶

    实践 入口 用来验证什么
    RAG 代码链路 完整链路代码实践 索引、检索、增强、生成怎样串起来
    ReAct Agent ReAct 代码实践 行动循环与 Observation 回填
    Harness 工程 Harness 代码实践 长任务环境、子任务与分发
    项目复盘 项目与代码实践总览 设计取舍、失败点、评测和面试表达
